# blending software「YouTuberSearch」
## エレベータピッチ
- このサービスは動画配信者が好きな人に向けた，あなたの**おすすめのチャンネル**を教えてくれます．
- YouTubeが公式で提供する**動画レベル**の推薦に比べて，意外性のある推薦結果が期待できます．
